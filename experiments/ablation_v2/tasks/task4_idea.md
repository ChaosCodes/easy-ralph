# Task 4: Research Idea - Multi-Agent Task Distribution

## User Request
"导师说：研究一下多 Agent 协作的任务分配策略"

## Ambiguity Points
- 什么是"好"的分配？
- 怎么协调？
- 有什么 baseline？
- 研究的是理论还是实现？

## Expected Clarifier Behavior

### Ask Mode (v1)
Problems:
- Student likely doesn't know specific answers
- Research direction is inherently exploratory
- Need to understand what's been done first

### Explore+Propose Mode (v2)
Should:
1. Search related work (multi-agent systems, task allocation)
2. Identify different research angles:
   - 任务分解策略（如何把大任务拆成子任务）
   - 负载均衡（如何分配给不同 agent）
   - 通信协调（agent 之间如何协作）
   - 冲突解决（多个 agent 同时修改怎么办）
3. Propose concrete research questions
4. Suggest evaluation methodology

## Pivot Scenario
This task is good for testing PIVOT_ITERATION:
- 如果第一个研究方向做不出有意义的结果
- 应该转向其他角度

## Success Criteria
- Clear problem formulation
- Literature context provided
- Specific research questions
- Evaluation plan
- Scope boundaries
